# CloneHTMLObj.html

Este repositorio contiene un archivo HTML llamado `CloneHTMLObj.html` que proporciona una funcionalidad para clonar elementos `<select>`.

## Descripción

El archivo `CloneHTMLObj.html` contiene un `<select>` inicial con opciones predefinidas. Cuando se hace clic en el botón "Clonar Select", se muestra un cuadro de diálogo que solicita un nuevo nombre para el `<select>` clonado. Después de ingresar el nombre, se crea una copia del `<select>` original y se inserta en un `<div>` destinado a contener los clones.



## Uso

1. Abre el archivo `CloneHTMLObj.html` en un navegador web.
2. Haz clic en el botón "Clonar Select".
3. Ingresa un nuevo nombre para el `<select>` clonado en el cuadro de diálogo.
4. Se creará una copia del `<select>` original con el nuevo nombre y se insertará en el `<div>` destinado, junto con un botón "Remover".
5. Repite los pasos 2 y 3 para clonar más `<select>` si es necesario.
